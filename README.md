# IMAG

It is an inventory management app mainly targetting stores.

## Features :
- [x] Produces Receipt
- [x] Management inventory
- [x] Management account
- [ ] Log

**Note:** Only admin have access to all page. *Regular* users have only acess to shop feature.

## How to use :
